# SurfaceChart

基于 ES 6 (ECMAScript 2015)  的 3D 曲面图表组件.

## 开发工具:WebStorm

## 源码编译

> $ cd SurfaceChart
// 安装编译工具
> $ npm install 
// 编译源码
> $ build.sh

##  编译依赖库 PIXI.js

> $ cd pixi

> $ npm install -g gulp

> $ npm install
// extras, loaders 目前这份代码去不掉
> $ gulp build --exclude extras --exclude filters --exclude loaders
